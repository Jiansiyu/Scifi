Hi John,

Attached is a zip file containing the pedestal fitting macro, calibration code, and block diagrams for both arms.  If you don't need the LHRS script, you can just ignore it (the RHRS script is more thoroughly commented anyways).  The pedestal fitting macro will produce a text file with pedestal values that will be read in by the calibration macro. 

The RHRS calibration uses the calorimeter block clusters built by the replay.  This could be modified to use tracks projected from the VDC to determine which block was hit (for this, see the LHRS script as an example).

I'm not familiar with APEX kinematics, but I imagine you might want to modify the cuts used to select good calibration events.  These are on lines 128-131 of RHRS_minuit_cluster_cut1.C.  Other than that, I think the only thing you should have to change is paths to ROOT files (in both pedestal and calibration scripts).

For best performance, the calibration macros should be run compiled:
root -b -q 'RHRS_minuit_cluster_cut1.C+'
 
Please let me know if you have any questions, or if the script isn't working.

Best,
Tyler
